# Exploratory Data Analysis & Business Intelligence

## Overview
This project performs exploratory data analysis and SQL-based business intelligence on a sales dataset.

## Folder Structure
- data/: Cleaned dataset
- eda/: Jupyter notebook with analysis and visualizations
- sql/: SQL queries for business questions
- results/: SQL outputs and insights

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Jupyter Notebook

## Key Business Questions
- What is the total revenue per year?
- Which products sell the highest quantity?
- What are the top revenue-generating products?

## Outcome
The analysis uncovers key trends and insights to support data-driven decisions.
